# Deployment

**Frontend:** Vercel  
**Backend:** AWS EC2 + PM2  
**Database:** RDS Postgres 16  
**CI/CD:** GitHub Actions  
**Secrets:** AWS Secrets Manager  
**Monitoring:** Grafana + Sentry

## Environment Variables

```bash
HIP3_RPC=https://rpc.hyperliquid.xyz
POSTGRES_URL=…  
API_KEY=…  
ORACLE_FEED_URL=https://oracle.hyperleverage.app
```

Auto-deploy on push to main. System halts trigger Discord alerts + SMS to ops.
